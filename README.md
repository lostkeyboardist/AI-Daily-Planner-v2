# AI-Daily-Planner-v2
AI Daily Planner: A Context-Aware Autonomous Agent 

A sophisticated Python-based daily organizer that goes beyond static scheduling. Leveraging the Gemini API and live Google Search grounding, this agent cross-references user tasks with real-time weather and local venue data to build optimized, human-readable itineraries. 

Built for people who want a schedule that actually makes sense for their day.
